# CSE 329 - Machine Learning

## Overview

CSE 329 covers machine learning theory and applications, with a strong focus on supervised, unsupervised, semi-supervised, and self-supervised learning. The course folder includes lecture notes for multiple instructors, term-final preparation files, recent TF solves, and special modules for semi-supervised learning.
