# CSE 321 - Computer Networks and Network Security

## Overview

CSE 321 covers computer networking, protocols, and network management. The folder combines lecture slides, instructor notes, books, class tests, and term-final preparation materials for different instructors.

## Useful Study Paths

### 1. Core Networking Topics
- Start with `Books/` and the Tanenbaum/Wetherall references
- Study `AAI sir/` notes for protocol theory and assignments
- Use `MSH sir/` slides for structured layer-by-layer networking

### 2. CT Preparation
- Check `all current resources/CT3/` for DLL and MAC materials
- Review `all current resources/CT 4/` for IoT and network fundamentals
- Use `CT-1 16 batchj.pdf` and `BUET_July_2025_CSE_321_CT_3__DLL___MAC__Question-1.pdf`

### 3. Term Final Preparation
- Study `CSE_321_TF_Questions_Merged.pdf`
- Use `TF prep 321_updatedNotes/TF prep 321/tf solve/` for solved TF papers
- Refer to `Razi sir handouts/` for TCP, congestion, transport layer, and network layer insights

### 4. Slide Notes by Instructor
- `slides_notes/1. razi sir/` - Razi Sir slides including TCP and transport layer
- `slides_notes/2. ananta sir/` - Data link layer, MAC, and physical layer notes
- `slides_notes/3. Shattik sir/` - IoT, HTTP, DNS, DHCP, and application layer notes
