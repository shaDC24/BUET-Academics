# CSE 311 - Data Communication & Signals Systems

## Overview

CSE 311 covers fundamental concepts in data communication and signal processing, including Fourier analysis, signal transformation, and communication system fundamentals. The course is taught by multiple instructors and includes both theoretical and practical components.

## Exam Preparation Strategy

### 1. **Understand Core Concepts**
   - Study Fourier fundamentals (Series, Transform, DFT, FFT)
   - Understand modulation techniques (AM, FM, PM, DM)
   - Learn signal processing operations

### 2. **Practice with Previous Papers**
   - Solve all CT papers in `Cts/` folder
   - Analyze CT solutions in `TF prep 311_updatedNotes/`
   - Review CT4 Makeup for additional practice

### 3. **Consolidate Knowledge**
   - Use RRM Suggestions and AYAS Sir notes for comprehensive coverage
   - Reference Oppenheim textbook for theoretical depth
   - Review formula sheets (সূত্র_311_Promi.pdf)

### 4. **Final Review**
   - Study merged TF questions
   - Use interactive Fourier resources (Asib Sir materials)
   - Focus on high-frequency topics from CT analysis
